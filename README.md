# NDYMAR-Keyboard
A keyboard layout designed only for fun. This is a layout that started as a joke in NSMB Central, but now turned as a functional layout. However, the NDYMAR keyboard is not as confortable as more popular layouts such QWERTY or AZERTY.

## Preview:

![Preview](/ndymar-preview.png)

# How to install the NDYMAR layout?

Simply download this repository and double click `setup.exe`, this will install the keyboard layout and it'll be ready to use on your device. If you want to uninstall it later, you can do so like uninstalling any other program.
